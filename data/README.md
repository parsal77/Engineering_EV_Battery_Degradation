# Data Directory

This project uses the NASA Ames Prognostics Center of Excellence (PCoE) Li-ion Battery Aging Dataset.

## Raw data (`data/raw/`)
- Expected files: `B0005.mat`, `B0006.mat`, `B0007.mat`, `B0018.mat`
- These files are intentionally excluded from git via `.gitignore`.

You can obtain the data from:
- NASA listing: https://data.nasa.gov/dataset/li-ion-battery-aging-datasets
- Kaggle mirror: https://www.kaggle.com/datasets/patrickfleith/nasa-battery-dataset

This repository also includes an automated downloader in `src/data_loader.py` that pulls the same files from a public mirror when you run preprocessing.

## Processed data (`data/processed/`)
Generated by the preprocessing and feature engineering pipeline:
- `{battery_id}_processed.csv` for each battery
- `all_batteries.csv` (merged discharge-cycle table)
- `features_all.csv` (feature-engineered modeling table)
